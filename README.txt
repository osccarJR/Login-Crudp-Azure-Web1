# Proyecto: Login y CRUD con ASP.NET Core MVC

## 🧠 Descripción
Este proyecto implementa un sistema de login con autenticación por cookies, y un CRUD completo para productos usando ASP.NET Core MVC (versión .NET 8.0) y Entity Framework Core. La aplicación se conecta a una base de datos SQL Server en Azure y ha sido desplegada exitosamente en Azure App Service.

---

## 📌 Características principales

- ✅ Sistema de Login con validación de credenciales.
- 🔐 Autenticación basada en cookies.
- 🔒 Sección CRUD protegida con [Authorize].
- 🧾 CRUD completo de productos (Create, Read, Update, Delete).
- 🗄️ Conexión a SQL Server en Azure mediante Entity Framework Core.
- 🧪 Seed inicial con un usuario: admin / 1234.
- 🎨 Interfaz visual con Bootstrap 5.
